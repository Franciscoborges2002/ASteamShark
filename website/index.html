Steam Shark

The tool that smells scam websites!

About

Steam Shark is an open source database to list scam and trust worthy website realated to steam. There are tree possible states:

  • > Trust worthy website
  • > Scam website
  • > Not registered website

If you enter an website that is not in database yet, you can register it. Know more how to register the website into the database here.
